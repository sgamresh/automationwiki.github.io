Appium setup on Mac OS for iOS Real device automation
Step 0:

Before proceeding please install JDK and download eclipse and follow below procedure.

JDK: Download JDK
ECLIPSE:Download Eclipse
Step 1:
First InstallHome-brew to manage a package in osX just copy paste the below command:
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
Step 2:
Install command line executer on mac using terminal by using below command:
xcode-select --install
Step 3:
Install carthage it is a part of os package command mentioned below:
brew install carthage
Step 4:
Install ruby before that just check ruby is installed or not by using ruby -v if not give below command:
\curl -sSL https://get.rvm.io | bash -s stable --ruby
Step 5:
Install node using terminal:
brew install node
Step 6:
Install Appium doctor:
npm install -g appium-doctor
Step 7:
Install appium through npm
npm install -g appium
Step 8:
Below command install missing packages for homebrew just copy past one by one:
1.brew tap qatools/formulas 2.brew install allure-commandline
Step 9:
How to set Java home path:
export JAVA_HOME=/Library/Java/Home
Step 10:
To install lib mobile devices follow below command:
brew install libimobiledevice
Step 11:
After all the steps we need to configure WebDriverAgentRunner in to our xcode. Go to the below path: open Application in mac and right on Appium and follow the path: /Applications/Appium.app/Contents/Resources/app/node_modules/appium/node_modules/appium-xcuitest-driver/WebDriverAgent/WebDriverAgent.xcodeproj
